Budget Reference Data

The reference data behind the Budget Builder. Use these tables independently when planning project costs, drafting grant applications, or benchmarking your own estimates.

All data on this page is generated from the same source as the Budget Builder tool. When we update our reference data, these tables update automatically.

Sources: Form 990 filings from GCR/EA sector nonprofits, 80,000 Hours job board salary ranges (2024-2026), EOR provider published pricing, and nonprofit operational benchmarks. All figures are approximate and should be verified for your specific context.
